Проект выполнен в рамках курса [Тестирование ПО:Автоматизация и Программирование.Python.Selenium](https://stepik.org/course/120491)

Для выполнения проекты выбран сайт https://www.onlinetrade.ru/

Выполнено тестирование основленого бизнес сценария (smoke test)

Структура проекта:
- base.base_class - в модуле находятся методы для проверки текущего url и его соответвия и метод для сохранения стриншотов страниц
- pages - в директории находятся модули страниц с олисанием локаторов и функций для взаимодействия с ними
- screen - в директорию сохряняются скриншоты успешных тестов
- tests - директория с тестами
